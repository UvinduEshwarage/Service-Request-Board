# Service Request Board (Frontend)

## Tech Stack
Next.js, React, Tailwind, Axios

## Setup

npm install

### .env.local
NEXT_PUBLIC_API_URL=http://localhost:5000/api

npm run dev

## Features
- View jobs
- Create job
- Filter jobs
- Update status
- Delete job

## Backend Required
Make sure backend is running on:
http://localhost:5000
